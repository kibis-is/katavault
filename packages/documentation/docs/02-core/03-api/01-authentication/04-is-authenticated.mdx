# isAuthenticated

> Checks if Katavault has been authenticated.

<SectionBreak />

## Usage

<Tabs
  defaultValue="main"
  values={[
    { label: 'main.ts', value: 'main' },
  ]}>
  <TabItem value="main">

    ```typescript
    import { Voi } from '@kibisis/chains';
    import { createKatavault } from '@kibisis/katavault-core';

    const katavault = await createKatavault({
      chains: [Voi],
    });
    const isAuthenticated = katavault.isAuthenticated();
    ```

  </TabItem>
</Tabs>

<SectionBreak />

## Returns

`boolean`

`true` if Katavault has been authenticated, `false` otherwise.
