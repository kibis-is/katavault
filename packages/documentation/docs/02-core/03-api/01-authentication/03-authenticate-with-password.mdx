# authenticateWithPassword

> Authenticates with the supplied password.

<SectionBreak />

## Usage

<Tabs
  defaultValue="main"
  values={[
    { label: 'main.ts', value: 'main' },
  ]}>
  <TabItem value="main">

    ```typescript
    import { Voi } from '@kibisis/chains';
    import { createKatavault } from '@kibisis/katavault-core';

    const katavault = await createKatavault({
      chains: [Voi],
    });

    await katavault.authenticateWithPassword({
      password: 'mtr*tpv!cbg@dhn9RFH',
      username: 'kibi@kibis.is',
    });

    console.log(`is authenticated: ${katavault.isAuthenticated()}`);
    /*
    is authenticated: true
    */
    ```

  </TabItem>
</Tabs>

<SectionBreak />

## Parameters

### params

* Type: [`AuthenticateWithPasswordParameters`](04-glossary/02-types.mdx#authenticatewithpasswordparameters)

The input parameters.

### params.password

* Type: `string`

The password to authentication with.

### params.username

* Type: `string`

A globally unique identifier for the user. This could be, for example, an email address.

<SectionBreak />

## Throws

### [DecryptionError](04-glossary/03-errors.mdx#decryptionerror)

If the stored password challenge from a previous authentication failed to be decrypted.

### [InvalidPasswordError](04-glossary/03-errors.mdx#invalidpassworderror)

If the supplied password does not match the password used to encrypt the stored password challenge.

### [PasswordTooWeakError](04-glossary/03-errors.mdx#passwordtooweakerror)

If this is a new sign-up, and the supplied password does not have enough entropy.
