# accounts

> Gets a list of the accounts within the wallet.

<SectionBreak />

## Usage

<Tabs
  defaultValue="main"
  values={[
    { label: 'main.ts', value: 'main' },
  ]}>
  <TabItem value="main">

    ```typescript
    import { createKatavault } from '@kibisis/katavault-core';

    const katavault = await createKatavault({
      user: {
        name: 'Kibi',
        username: 'kibi@kibis.is',
      },
    });
    const accounts = await katavault.accounts();

    console.log(accounts);
    /*
    [
      {
        address: 'SH2KEL633QIJLURBOOJTIKR5EIIYR4A2VYZC2DB6ZD4ZZL6XHYFBETEOGU',
        name: 'Account-1',
      },
      {
        address: 'NWANMINCOFDOHXHMXUTP76IOUZQFYHJUYGHSCPOCMX3JHZCC2MWXTZNUAI',
        name: 'Account-2',
      },
    ]
    */
    ```

  </TabItem>
</Tabs>

## Returns

[`Promise<Account[]>`](04-glossary/02-types.mdx#Account)

A promise that resolves to the list of accounts stored in the wallet.
