# generateAccount

> Generates a new account in the wallet.

<SectionBreak />

## Usage

<Tabs
  defaultValue="main"
  values={[
    { label: 'main.ts', value: 'main' },
  ]}>
  <TabItem value="main">

    ```typescript
    import { type Account, createKatavault } from '@kibisis/katavault-core';

    const katavault = await createKatavault();
    let account: Account;

    await katavault.authenticateWithPasskey({
      user: {
        name: 'Kibi',
        username: 'kibi@kibis.is',
      },
    });

    account = await katavault.generateAccount('Personal');

    console.log(account);
    /*
    {
      address: 'SH2KEL633QIJLURBOOJTIKR5EIIYR4A2VYZC2DB6ZD4ZZL6XHYFBETEOGU',
      name: 'Personal',
    }
    */
    ```

  </TabItem>
</Tabs>

<SectionBreak />

## Parameters

### name (optional)

* Type: `string`
* Default: `undefined`

A name for the account.

<SectionBreak />

## Returns

[`Promise<Account>`](04-glossary/02-types.mdx#account)

A promise that resolves to the created account.

<SectionBreak />

## Throws

### [EncryptionError](04-glossary/03-errors.mdx#encryptionerror)

If the account's private key failed to be encrypted.

### [NotAuthenticatedError](04-glossary/03-errors.mdx#notauthenticatederror)

If the provider has not been authenticated.


### [FailedToAuthenticatePasskeyError](04-glossary/03-errors.mdx#failedtoauthenticatepasskeyerror)

If the passkey did not return the encryption key credentials.

### [FailedToRegisterPasskeyError](04-glossary/03-errors.mdx#failedtoregisterpasskeyerror)

If, when attempting to register a passkey, the public key credentials failed to be created on the passkey.

### [PasskeyNotSupportedError](04-glossary/03-errors.mdx#passkeynotsupportederror)

If the browser does not support WebAuthn or the passkey does not support storing encryption key material.

### [UserCanceledPasskeyRequestError](04-glossary/03-errors.mdx#usercanceledpasskeyrequesterror)

If the user canceled the passkey register/authenticate request or the request timed out.
