# removeAccountByKey

> Removes the account from the vault for the specified key if it exists.

:::note

Authentication is **REQUIRED**.

:::

<SectionBreak />

## Usage

<Tabs
  defaultValue="main"
  values={[
    { label: 'main.ts', value: 'main' },
  ]}>
  <TabItem value="main">

    ```typescript
    import { Voi } from '@kibisis/chains';
    import { createKatavault } from '@kibisis/katavault-core';

    const katavault = await createKatavault({
      chains: [Voi],
    });

    await katavault.authenticate();
    await katavault.removeAccountByKey('CZepAkxcz4KEruVyrURwGFh8SEkKLFL8nASuXmS2Zu3a');
    ```

  </TabItem>
</Tabs>

<SectionBreak />

## Parameters

### key

* Type: `string`

The <a href={BASE_58_SPECIFICATION_URL} rel="noopener noreferrer" target="_blank">base58</a> encoded public key of the account to remove from the vault.

<SectionBreak />

## Throws

### [NotAuthenticatedError](04-glossary/03-errors.mdx#notauthenticatederror)

If Katavault has not been authenticated.
