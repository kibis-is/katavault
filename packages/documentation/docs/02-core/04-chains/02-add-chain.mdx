# addChain

> Adds the new chain to the list of supported chains if it does not already exist, otherwise it updates the existing chain by the <a href={CAIP_002_SPECIFICATION_URL} rel="noopener noreferrer" target="_blank">CAIP-002</a> chain ID.

<SectionBreak />

## Usage

<Tabs
  defaultValue="main"
  values={[
    { label: 'main.ts', value: 'main' },
  ]}>
  <TabItem value="main">

    ```typescript
    import { Algorand, Voi } from '@kibisis/chains';
    import { createKatavault } from '@kibisis/katavault-core';

    const katavault = await createKatavault({
      chains: [Voi],
    });

    await katavault.addChain(Algorand);
    ```

  </TabItem>
</Tabs>

<SectionBreak />

## Parameters

### chain

* Type: `ChainConstructor`

The chain to be added.
