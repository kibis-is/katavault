# Installation

> Install the React implementation of Katavault using a package manager or via a CDN using a `<script>` tag.

<SectionBreak />

## Package manager

Install the required packages.

```bash npm2yarn
npm install algosdk@3.x react@18.x @kibisis/katavault-core @kibisis/katavault-react
```

<SectionBreak />

## CDN

If you're not using a package manager, you can install it via an ESM-compatible CDN such as esm.sh. Add a `<script type="module">` tag to the bottom of your HTML file with the following content.

```html
<script type="module">
  import 'https://esm.sh/algosdk';
  import 'https://esm.sh/react';
  import 'https://esm.sh/@kibisis/katavault-core';
  import { KatavaultProvider } from 'https://esm.sh/@kibisis/katavault-react';
</script>
```
