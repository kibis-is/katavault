# useSetAccountNameByKey

> Hook to set the name for an account by its key.

:::note

Authentication is **REQUIRED**.

:::

          name: 'New name',
        }, {

          },

## Returns

`(params: SetAccountNameByKeyParameters, options?: { onError, onSuccess }) => void`

The function can be called with callbacks that are invoked on error/success.

#### options.onError

Type: `(error: BaseError, params: SetAccountNameByKeyParameters) => void | Promise<void>`

This will be invoked if there was an error.

#### options.onSuccess

Type: `(result: Account, params: SetAccountNameByKeyParameters) => void | Promise<void>`

This will be invoked if the account update was successful and will return the updated account.
