# useSignMessage

> Hook to sign a message or some arbitrary bytes using the appropriate account and chain.

:::note

Authentication is **REQUIRED**.

:::

    ```typescript
    import { useAccounts, useSignMessage } from '@kibisis/katavault-react';
    import { useCallback } from 'react';

    function SignMessage() {
      const accounts = useAccounts();
      const signMessage = useSignMessage();
      const handleOnSignMessageClick = useCallback((key: string) => () => {
        signMessage({
          accountKey: key,
          chainID: 'avm:r20fSQI8gWe_kFZziNonSPCXLwcQmH_n',
          encoding: 'hex',
          message: 'hello humie!',
        }, {
          onError: (error) => {
            console.error(error);
          },
          onSuccess: (result: string) => {
            console.log(`signed message with signature: ${result}`);
          },
        });
      }, [signMessage]);

      return (
        <div>
          {accounts.map(({ key, name }) => (
            <div>
              <p>Key: {key}</p>
              <p>Name: {name ?? '-'}</p>
              <button onClick={handleOnSignMessageClick(key)}>Sign</button>
            </div>
          ))}
        </div>
      );
    }
    ```

## Returns

`(params: SignMessageParameters, options?: { onError, onSuccess }) => void`

The function can be called with callbacks that are invoked on error/success.

#### onError

`(error: BaseError, params: SignMessageParameters) => void | Promimse<void>`

This will be invoked if there was an error.

#### onSuccess

`(result: string | Uint8Array, params: SignMessageParameters) => void | Promimse<void>`

This will be invoked with the signature as the result if the message signing was successful.

:::note

If the `params.encoding` is used, the result will be a string in the specified encoding. Otherwise, the raw signature bytes (`Uint8Array`) will be returned.

:::
