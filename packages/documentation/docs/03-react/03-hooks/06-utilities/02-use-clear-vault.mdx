# useClearVault

> Hook to perform a "factory reset." This will remove authentication credentials, settings and all account data.

:::note

Authentication is **REQUIRED**.

:::

:::caution

Use with extreme caution. This is action is irreversible.

:::

<SectionBreak />

    ```typescript
    import { useClearVault } from '@kibisis/katavault-react';
    import { useCallback } from 'react';

    function ClearVault() {
      const clearVault = useClearVault();
      const handleOnClearVaultClick = useCallback(() => {
        clearVault({
          onError: (error) => {
            console.error(error);
          },
          onSuccess: () => {
            console.log(`vault cleared!`);
          },
        });
      }, [clearVault]);

      return (
        <div>
          <button onClick={handleOnClearVaultClick}>Clear Vault</button>
        </div>
      );
    }
    ```

## Returns

`(options?: { onError, onSuccess }) => void`

The function can be called with callbacks that are invoked on error/success.

#### options.onError

Type: `(error: BaseError) => void | Promise<void>`

This will be invoked if there was an error.

#### options.onSuccess

Type: `() => void | Promise<void>`

This will be invoked if the action was successful.
